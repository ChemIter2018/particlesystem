# 交互式粒子系统

一个使用 Vue 3、Three.js 和 MediaPipe Hands 构建的实时交互式 3D 粒子系统。

## ✨ 特性

### 🎮 增强的手势控制
- **张开/合拢手掌**：控制粒子群的缩放大小
  - 手掌完全张开 = 粒子放大到最大
  - 手掌握成拳头 = 粒子缩小到最小
- **左右移动手**：控制粒子群的水平旋转
  - 手向右移 = 粒子向右旋转
  - 手向左移 = 粒子向左旋转
- **上下移动手**：控制粒子群的垂直翻转
  - 手向上移 = 粒子向上翻转
  - 手向下移 = 粒子向下翻转
- **动态颜色**：粒子颜色会根据手的张开程度实时变化（青色到洋红色渐变）

### 🎨 精美的用户界面
- **玻璃态设计**：使用 glassmorphism 效果的现代化界面
- **实时摄像头预览**：带手势骨骼可视化的摄像头画面
- **手势提示面板**：左上角显示手势控制说明和检测状态
- **实时数据面板**：右上角显示缩放、旋转、倾斜角度等实时数据
- **流畅动画**：所有 UI 元素都有精心设计的进入和悬停动画

### 🌟 多种粒子形状模板
- 心形 ❤️
- 花朵 🌸
- 土星 🪐
- 佛像 🧘
- 烟花 🎆

### ⚙️ 可调节参数
- 粒子数量（500-10000）
- 粒子大小
- 粒子颜色
- 手势灵敏度调节：
  - 缩放灵敏度
  - 旋转灵敏度
  - 翻转灵敏度
- 摄像头显示开关

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 运行开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

## 📖 使用指南

1. **允许摄像头权限**：首次运行时，浏览器会请求摄像头访问权限，请允许。

2. **手势控制**：
   - 将一只手放在摄像头前
   - 确保手势提示面板显示"✓ 检测到手势"
   - 尝试张开/合拢手掌来缩放粒子
   - 左右移动手来旋转粒子
   - 上下移动手来翻转粒子

3. **调整设置**：
   - 点击右上角的控制面板
   - 在"粒子形状"文件夹中更改模板和颜色
   - 在"手势控制"文件夹中调整灵敏度
   - 在"显示设置"中切换摄像头显示

## 🎯 技术栈

- **Vue 3** - 组合式 API
- **Three.js** - 3D 渲染引擎
- **MediaPipe Hands** - 手势识别
- **lil-gui** - 控制面板
- **Vite** - 构建工具

## 💡 提示

- 在光线充足的环境中使用效果最佳
- 确保手与摄像头保持适当距离（约 30-60cm）
- 如果检测不稳定，可以在控制面板中调整灵敏度
- 手势控制支持单手操作，更加方便自然

## 🎨 设计亮点

- **现代化 UI**：使用 Inter 字体和精心设计的配色方案
- **响应式设计**：支持桌面和移动设备
- **流畅交互**：所有手势控制都有平滑的过渡效果
- **视觉反馈**：实时显示手势状态和参数数值
- **深色主题**：优雅的深色渐变背景
